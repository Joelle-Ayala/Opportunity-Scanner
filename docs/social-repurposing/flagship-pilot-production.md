## Autonomous Video Stack

### Required

1. Descript API or MCP connection tied to the Opportunity Scanner drive.
2. Higgsfield connection with an approved credit ceiling.
3. Business-controlled storage for final MP4, source project link, captions, thumbnail, and publishing copy.

### Already Available

- Final scripts, scene plans, claims, caveats, CTA copy, and brand assets.
- Canva connection for thumbnails and channel graphics.
- Higgsfield generation, stock text-to-speech, assembly, and caption tools.
- Repository storage and public delivery through Opportunity Scanner.

### Not Required

- A cloned founder voice.
- A founder avatar or recorded presenter footage.
- Adobe Premiere or another manual desktop editor.
- Another standalone text-to-speech vendor.

Descript's API can create and edit projects, run Underlord edits, publish a finished project, and return signed media URLs. It currently does not offer a direct local export endpoint, so the automated workflow should use Descript publishing or download the signed MP4. Source: https://help.descript.com/hc/en-us/articles/43370311322509-Descript-API

## Approval Boundary

After the initial integrations, the standing operating default can be:

- Use a stock voice and non-presenter motion graphics.
- Do not clone a voice or likeness without separate approval.
- Generate within the approved monthly credit ceiling.
- Keep claims grounded in the source article or lead magnet.
- Save the MP4, captions, thumbnail, copy, source links, and cost in the production queue.
- Publish only after the account connections and channel-level publishing approval are established.
